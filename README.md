# Blog con React y API Rest en NodeJS

Este proyecto consiste en un blog con una interfaz de usuario construida en React y una API Rest desarrollada en NodeJS con Express. El backend y frontend están separados para facilitar la escalabilidad y el mantenimiento del código.

## Backend (API Rest en NodeJS)

### Configuración

- **Nombre del Proyecto:** api-rest-node
- **Versión:** 1.0.0
- **Autor:** Matias Daniel Delgado

### Dependencias Principales

- Express: Framework web para NodeJS
- Mongoose: Biblioteca de modelado de objetos MongoDB para NodeJS
- Multer: Middleware para gestionar la carga de archivos
- Cors: Middleware para habilitar CORS en Express
- Dotenv: Carga de variables de entorno desde un archivo
- Validator: Biblioteca para validación de datos
- Nodemon: Herramienta que reinicia automáticamente la aplicación durante el desarrollo

### Scripts

- `start`: Inicia la aplicación con nodemon

### Instalación

```bash
cd api-rest-node
npm install

npm start
```

# Proyecto: Blog

## Descripción

Este proyecto es una aplicación simple de blog construida con React. Utiliza Vite como servidor de desarrollo y herramienta de construcción, junto con React Router para manejar la navegación. El código se configura con ESLint para el linting y asegurar la calidad del código.

## Empezar

1. Clonar el repositorio:

   ```bash
   git clone <url-del-repositorio>
   npm install
   npm run dev
   La aplicación estará disponible en http://localhost:3000.
   ```
